# Agent Instructions for Vändtia

This repository contains a Flutter implementation of the card game "Vändtia".

## Documentation
- See [RULES.md](RULES.md) for the detailed game rules.

## Tech Stack
- Flutter
- Riverpod v3 (State management)
- Flame (Game engine)

## Coding Standards
- Follow Clean Architecture.
- Keep domain logic (rule engine) independent from Flame.
- Use immutable models for state.
- Use Riverpod Notifiers for state management.
